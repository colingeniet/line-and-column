§ colorPair

enum colorPair

Colors pairs.

Create all macro required by ncurse to describe colors. Every square's color in the grid is define by a pair with blanck caracter writed on a colord backfont. Color pair 0 is reserved by ncurses as default color pair it is initialized by start_colors() and should not be modified.
